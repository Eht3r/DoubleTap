## 현재 구현된 기능
1. 폴더 추가: 폴더 추가 버튼을 누르면 입력창이 뜨고 이름을 넣으면 폴더가 추가됨
2. 생성한 폴더 접근: 안드로이드 파일관지라앱(Files)에서 접근 가능
3. 폴더 추가 버튼으로 폴더를 생성하지 않고 다른 방식으로 폴더 수정시(생성, 이름변경, 삭제) 바로 반영이 안되므로 위에서 아래로 드레그시 반영
